Product OverviewFMSI OneCX (Previously known as Banking Connections/ BC)
3
Core Objective:To unify and consolidate the functionalities of two separate legacy products—FMSI Lobby (LT) and FMSI Appointment (AC)—into a single, integrated platform.
4
Primary User Segments:1. Staff: Admin Portal users (System Admin, Client Admin, Executive, Managers, Branch Managers, Staff Members).
2. Customers: End-users using Web Widget, Remote Check In portal and Kiosk.
5
Key Value Propositions• Unified Centralized Management: OneCX eliminates the need to duplicate work across multiple systems by centralizing critical administrative tasks
• Granular Configurability: The platform offers highly customizable settings, such as "Kiosk Configurations" that can be unique to specific locations and "Permission Groups" that allow for detailed control over user access rights.
• AI integration: Enable Glia Virtual Assistants in the Web Widget
