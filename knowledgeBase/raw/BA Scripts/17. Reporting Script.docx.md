### **Accessing the Lobby Activity Report**

To begin, navigate to the ‘Reports’ section on the left-hand menu, and click on ‘Lobby Activity’. This opens the ‘Lobby Activity’ reporting dashboard where you can track key performance metrics, analyze trends, and drill into user behavior.

### **Report Views**

The ‘View’ dropdown is where the real power lies. Depending on the view you select, the report below will adjust its layout and columns to suit the analysis type:

1. **Location Comparison**  
   A side-by-side comparison of branches or regions — showing average wait times, number of visitors assisted, products/services provided and average assist time. Ideal for benchmarking performance across locations.  
2. **Traffic & Wait Time**  
   Provides insights on daily traffic volume and corresponding wait times based on hour of day, day of week, week of month and by month. Helps identify peak days and service bottlenecks.  
3. **Products & Services**  
   Focuses on which services are being provided the most, the service count and the average assist time based on service types. A great view for understanding customer needs and service demands.  
4. **Employee**  
   Tracks service delivery by staff — showing how many products/services were provided by each employee and their average assist time. Helps with individual performance evaluation.  
5. **Details**  
   The most granular view — listing every interaction with lobby visitors in detail, including visitor information, interaction details, services provided, wait and assist durations, and status updates.  
6. **Staff Signed In/Out**  
   Displays sign-in and sign-out data of staff users per location, including assist time, idle time, and signed-out reason. Useful for tracking user presence and time spent.

### **Drill Down and Export**

You can expand each row to reveal detailed interactions, especially useful in views like ‘Details’ and ‘Staff Signed In/Out’. Once you’ve reviewed the data, click on the ‘Export’ button to download the report in Excel format for further analysis or sharing.

# **Filter Options**

By default, the ‘Filter’ dropdown is set to ‘None’. But clicking the ‘Filter Options’ button will open up the ‘Filter Options’ popup, allowing you to create detailed filtering rules across the following sections:

* **Where & When**  
  * Locations: Select one or more locations to view records from.  
  * Day of the Week: Filter based on the days when interactions occurred.  
      
* **Staff**  
  * Requested Staff: View records where certain staff members were requested.  
  * Supervisor: Filter records involving supervisory or non-supervisory staff.  
  * Assisted By: See which staff assisted visitors.  
      
* **Products & Services**  
  * Category: Select one or more categories.  
  * Products & Services: Specify which services were provided.  
      
* **Other Filters**  
  * Status: Filter by status such as Completed, Left Without Assistance, or No Show.  
  * Auto-Wrap Up Records: Choose to include or exclude records that were auto-wrapped.  
  * Assist Type: View ‘Standard’ or ‘Instant’ assists.

You can save your filters as reusable Templates using the ‘Save’ or ‘Save As’ buttons. If a template is checked as Default, it will be auto-applied when you open the report. You can also share templates with other users by checking the ‘Share Template’ option.

**Lobby Editor Report**  
To access the Lobby Editor, click on ‘Lobby Editor’ located under the ‘Reports’ section. This will open up the Lobby Editor report page, where you’ll see a date filter on the top right to define your report range. You can enter the ‘From’ and ‘To’ dates and click ‘Run Report’ to view the records for that period.

# **Lobby Editor Grid Overview**

The main grid presents a list of all lobby records that match the specified filters. Each row displays key data such as:

* Lobby ID  
* Check-In Time  
* Location  
* Account Number  
* Account Name  
* Confirmation Number  
* Requested Products/Services  
* Requested Staff  
* Wait Time  
* Assist Time  
* Wrap-Up Time  
* Interaction Time  
* Auto Wrap-Up  
* Status  
* Action icons (including an edit icon)

To view more detailed information for a particular record, click the expand icon (+) at the start of that row. This will reveal the products/services provided during the interaction, whether each one was requested or not, reference numbers, who assisted the visitor, and the timing details.  
You can also expand or collapse all records at once using the "Expand All" button at the top of the grid.  
If needed, you can delete selected records in bulk using the checkbox on the left of each row and the Delete Selected button.

### **Filters and Templates**

At the top, you’ll see options to select a Filter Template and define your date range. By default, the template is set to ‘None’, but clicking on the Filter Options button opens the filter popup — this reflects the same functions and capabilities as you might have seen in the Lobby Activity report.

# **Editing a Lobby Record**

Each record can be edited by clicking on the pencil icon in the ‘Action’ column. This opens up the ‘Edit Record’ popup, which contains several key sections:

* **Customer Information**  
  * Includes editable fields for Account Number, First Name, Last Name, Email Address, Phone Number, and any custom fields configured for your organization.  
      
* **Interaction Information.** Fields such as:  
  * Location  
  * Added By (who checked in the record)  
  * Lobby ID and Confirmation \#  
  * Status (Completed, No Show, LWA, etc.)  
  * Product/Service Requested (can select one or multiple)  
  * Staff Requested  
  * Check-In Time  
  * Completed Time  
  * Notes (optional)  
      
* **Product/Services Provided.** This section breaks down the interaction(s) where services were rendered. Each interaction is listed with:  
  * Product/Service  
  * Category  
  * Reference \#  
  * Assisted By  
  * Assist Started Time  
  * Wait Time  
  * Assist Time  
  * Wrap-Up Time  
  * Total Interaction Time

If a record had multiple interactions — for example, if a visitor was assisted, returned to the lobby, and taken again — each interaction will appear sequentially.  
You can click the "Add Product/Service" button to add new records if required and use the trash icon to remove them.  
Once all changes are made, click ‘Save’ to update the record and click ‘Cancel’ to discard changes.

# **Exporting the Report**

If you’d like to work with this data offline, you can click the ‘Export’ button to download the full report based on the current filter settings.
