## **Users** 

To access the Users module, navigate to the User Management section and click on "Users".  
Within the Users module, you will see:

* An “Add User” button to add new users.  
* A search option to locate specific users.  
* A permission group filter to filter user records based on a specific permission group.  
* An active status filter for active/inactive users.  
* A location selector that defaults to the logged-in user's home location, showing sub-locations as options.  
* A checkbox labeled “Include Sublocations” – when checked, users from both home and sub-locations are shown.

In the grid, user details such as user code, username, first name, last name, home location, permission groups, and notification groups are displayed.

**Creating or Editing a User**    
Whether you're creating a user via the “Add” button or editing an existing one through the edit icon, the layout is the same.  Here, we follow a task-based UI model in OneCX. Settings are grouped into sections: Basic Settings, Lobby Settings, Appointment Settings, Availability Template, and Skills.

**Basic Settings Section:**  
In addition to the basic personal information, some of the key data fields displayed in this section include:

* Email address \- used as username  
* Permission groups \- determines feature access  
* Notification groups \- for alert triggering – set under Permission Groups module  
* Primary Notification Email \- for alerts, appointment notices, etc.  
* Phone number and preferred language (English/Spanish) selections are optional  
* Home location – determines the logged-in user's accessible locations hierarchy  
* And then we have Active status, Spanish speaker, account locked, login enabled, two-factor authentication  
  * Login Enabled is only applicable for Appointment Concierge-enabled clients. For Lobby Tracker licensed clients, only the Active status is required for login.

When saving a new user for the first time, we have two options as ‘Save’ and ‘Save and Continue’

* Clicking on ‘Save’ navigates back to the user grid. The user appears in the main grid.

* Clicking on ‘Save and Continue’ retains the same view allowing you to edit other settings after saving the user.

When closing the section, click “Close” to minimize the expanded section  

**Lobby Settings Section – visible only for Lobby Tracker licensed clients:**

* Includes options that control the related user’s ability to be displayed in lobby and kiosk  
* Allows to set supervisor selection or marking as eligible supervisor  
* The ‘Default’ view when accessing the ‘Lobby Queue’ page can be selected from the list of tab options such as Lobby Staff Status, Appointment Status, Central Views like Lobby, Staff and Location.  
* Impersonation related settings handle the capability for the logged in user to perform lobby assistance by impersonating another staff member and the eligibility for themselves to be impersonated by another user.

**Appointment Settings Section – visible only for Appointment Concierge licensed clients:**

* Includes options that control time zone and secondary notification email (optional) settings.  
* The subscription to daily/summary emails and enabling flexible float capabilities   
* The ‘Appointments’ option must be checked to schedule appointments and the ‘Use Events” option must be checked for the user to be displayed for events.  
* Finally, the capability to restrict user being scheduled outside operating hours and meeting preferences can be set.

**Availability Template Section:**

* Shows user's available days, time slots, and default work locations (can be modified).  
* Allows to set multiple rotations.  
* Meeting preferences can be enabled via the ‘Appointment Settings’ section and set for the user at a day level in this view.

**Skills Section**

1. Allow to select and opt-in for products and services by category.  
2. The proficiency per skill as be adjusted as well between 0 – 100\.

**Saving a Section** – Click on “Save” to update changes done in the ‘Lobby Settings’, ‘Appointment Settings’, ‘Availability Template’ or ‘Skills’ sections. 

**Closing a Section** – Click “Close” to minimize the expanded section

**My Account**  
To access the ‘My Account page, click the logged in user’s name in the top right corner and select ‘My Account’ in the dropdown.

This view allows the logged in user to edit their personal information. 

Editable fields in the page include:

* First name and Last name, preferred time zone, primary and secondary notification email, phone number, preferred language, lobby default tab  
* You can also change your password through the ‘My Account’ page or use "Forget Password" in the login to receive a reset email

Non-editable fields in the page include:

* Username, permission groups, notification groups, supervisor assignments

**For saving** \- click on “Save” to update changes in the ‘My Account’ page. 

**For cancelling** \- click on “Cancel” to discard any changes.

And that’s a quick walkthrough of the Users and My Account modules in OneCX. These tools are designed to give you complete control over how your system is configured and presented to users.
