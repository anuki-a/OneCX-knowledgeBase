## **Data Import Tab**

Let’s begin with Data Import/Export — found under the Configuration section in the left-hand menu.  
The default tab is Data Import, where you can upload data in bulk using CSV files.

Start by selecting the type of file you wish to import from the dropdown — options include Locations, Holidays, Operating Hours, Product Categories, Products & Services, User Availability, User Skills, and Users.  
\[Hover over the dropdown. Show a few options being selected.\]

Once you’ve selected the file type, upload your file using either the ‘Select File’ button or simply drag and drop it into the upload area.  
\[Show file being uploaded.\]

Then, hit ‘Submit File’ to initiate the import process.  
\[Click “Submit File.”\]

## **View Import Logs**

After uploading, you can review the ‘Import Logs’ below.  
These logs provide detailed feedback on the success or failure of each record in your upload — making it easier to track errors and make corrections.  
\[Reveal log area and highlight successful and failed rows.\]

## **Data Export Tab**

Switching to the Data Export tab — here, you can easily download system data in CSV format.  
\[Switch to the tab.\]

Choose the file type — such as Users, Locations, or Products & Services — and click the Data Export button. This allows you to retrieve and reuse system data whenever needed.  
\[Show dropdown and click “Data Export.”\]

## **Localization Module**

Next, let’s look at the Localization module — located just below Data Import/Export in the side menu panel. This powerful tool allows you to customize how text appears across the OneCX platform in different applications and languages. 

* From the Application dropdown, you can select where the text appears — like FMSI OneCX, Kiosk, or the Remote Check-In Web Page.  
  \[Highlight the Application dropdown.\]

* Use the Language selector to switch between English and Spanish translations.  
  \[Switch to Spanish.\]

* The grid shows system text keys along with their default English values and customized display text.  
  \[Scroll through the table briefly.\]

## **Editing Localization Text**

* To modify a translation, click the ‘Edit’ icon beside any text row.  
  \[Click the pencil icon.\]

Here, you can edit the English or Spanish text depending on the selected language, then click ‘Save’ to apply changes instantly or ‘Cancel’ to discard changes.  
\[Demonstrate editing and saving.\]

## **Wrap-up**

To customize the appointments related text, you can access the ‘Localization’ tab under ‘Administration’ module which will be displayed once you click on the ‘Appointments’ main menu option. And that’s a quick walkthrough of the Data Import/Export and Localization modules in OneCX. These tools are designed to give you complete control over how your system is configured and presented to users.
